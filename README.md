# Push_swap

## Descripción

Push_swap es un proyecto de algoritmia en C que ordena datos utilizando dos stacks y un conjunto limitado de instrucciones. El objetivo es escribir un programa que encuentre la solución más eficiente para ordenar números enteros.

## Requisitos

- Lenguaje: C
- Normas: Se debe seguir la norma de programación.
- Archivos a entregar: `Makefile`, `*.h`, `*.c`
- Funciones autorizadas: `read`, `write`, `malloc`, `free`, `exit`.

## Instrucciones

1. Clona este repositorio.
2. Compila el proyecto usando `make`.
3. Ejecuta el programa con la lista de números como argumento.

## Ejemplo de uso

```bash
./push_swap 5 2 3 1 4
```
