# Push_swap: Algorithmic Sorting in C
## Description
Push_swap is an advanced algorithmic project in C that challenges optimization skills by sorting data using two stacks and a limited set of instructions. The objective is to implement an algorithm that efficiently solves the problem of sorting a list of integers while minimizing the number of operations.
This project is ideal for those looking to deepen their understanding of data structures, memory management, and algorithmic efficiency in C.

## Project Requirements
Language: C
Standards: Must adhere to the specified coding norms.
Deliverable Files: Makefile, *.h, *.c
Allowed Functions: read, write, malloc, free, exit.
## Usage Guide
Follow these steps to run the project:
1. Clone the repository:
```
git clone https://github.com/your_username/push_swap.git
```
2. Compile the project using the make command:
```
make
```
3. Run the program with a list of numbers as an argument:
```
./push_swap 5 2 3 1 4
```
The program will generate the necessary operations to sort the numbers in the least amount of steps.

## Project Structure
The code is organized into several sections, covering different operations and manipulations of the stacks. Below is a guide to the abbreviations used in the project for easier navigation and understanding:

### Glossary of Abbreviations (Files)
0__PS_MF: Push_swap Main Functions
1_PS_SH: Stack Handling
2_PS_CMP: Push_swap Complementary Functions
3_PS_BSO: Basic Stack Operations
4_PS_Args: Argument Handling
### Glossary of Abbreviations (Code)
BSO: Basic Stack Operations
AORB: A or B stack operations
ACOST: Cost associated with stack A
BCOST: Cost associated with stack B
PS: Push_swap
SH: Stack Handling
CMP: Complementary
MF: Main Functions
TEMP: Temporary variables
LIBFT: Standard C functions from the libft library
```
int AORB -> a = 0 && b = 1
  // Example for PA (Push to A)
  dst = t_stack **a;
  src = t_stack **b;
  // Example for PB (Push to B)
  dst = t_stack **b;
  src = t_stack **a;
```
These operations enable efficient movement of elements between stacks and the necessary manipulations for sorting.
